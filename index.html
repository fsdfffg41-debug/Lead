How it works

From search to leads in
under 60 seconds

MapLeads automates the entire discovery pipeline — no API keys needed, no rate limits, no setup friction.

STEP 01
🗺️

Enter your search

Type any business category and location. We support city, region, postal code, or country-level searches across 30+ countries.

STEP 02
🔎

We scrape Google Maps

Our engine fetches live Google Maps results, bypassing restrictions without any API key. Always fresh, never cached from months ago.

STEP 03
🚫

Filter: no website only

We detect which businesses are missing a website — a massive untapped market for web agencies, SaaS tools, and outreach campaigns.

STEP 04
📤

Export clean data

Download as CSV or JSON with name, phone, address, category, rating, review count, GPS coordinates, and website status.

Developer API

REST API. Clean endpoints. Zero headaches.

Integrate MapLeads into your stack in an afternoon. Full OpenAPI spec, SDKs for Node, Python, and PHP, and response times under 2s.

  • GET /v1/search Search businesses
  • GET /v1/business/{id} Get full details
  • POST /v1/bulk Start bulk scrape job
  • GET /v1/jobs/{id} Check job status
  • GET /v1/export/{id} Download results
  • POST /v1/webhooks Register webhook
● Example request 
curl -X GET \
  "https://api.mapleads.io/v1/search
  ?query=restaurants
  &location=Berlin%2C+Germany
  &filter=no_website
  &limit=100" \
  -H "Authorization: Bearer YOUR_API_KEY"

API Documentation

Complete reference for integrating MapLeads into your application. Fast, reliable, and easy to use.

Introduction

MapLeads API provides real-time access to business data from Google Maps. Scrape businesses without websites, get verified phone numbers, and export clean data in seconds.

Base URL: https://api.mapleads.io/v1

Authentication: Bearer token in Authorization header

Authentication

All API requests require an Authorization header with a bearer token:

curl https://api.mapleads.io/v1/search \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json"

Get your API key from your admin dashboard.

Quick Start

1. Install SDK

npm install mapleads

2. Initialize Client

const MapLeads = require('mapleads'); const client = new MapLeads('YOUR_API_KEY');

3. Search Businesses

const results = await client.search({ query: 'restaurants', location: 'Berlin, Germany', filter: 'no_website' });

Search Endpoint

GET
/search

Search for businesses matching criteria

Parameters

Parameter Type Required Description
query string Yes Business category (e.g., restaurants, plumbers)
location string Yes City, region, or country
filter string No no_website, has_phone, rating_4plus
limit integer No Results per page (default: 50, max: 500)

Example Request

curl "https://api.mapleads.io/v1/search?query=restaurants&location=Berlin,Germany&filter=no_website&limit=100" \ -H "Authorization: Bearer YOUR_API_KEY"

Example Response

{ "status": "success", "total": 2847, "results": [ { "name": "Gasthaus Zur Linde", "phone": "+49 30 2456 7890", "address": "Lindenstr. 14, Berlin", "rating": 4.3, "has_website": false, "lat": 52.5061, "lng": 13.3954 } ] }

Business Details

GET
/business/{id}

Get detailed information about a specific business

Parameters

Parameter Type Description
id string Business ID from search results

Bulk Jobs

POST
/bulk

Start a bulk scraping job

curl -X POST https://api.mapleads.io/v1/bulk \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "query": "restaurants", "location": "Berlin, Germany", "filter": "no_website" }'

Export Data

GET
/export/{id}

Download results as CSV or JSON

Supported formats: CSV, JSON, Excel
